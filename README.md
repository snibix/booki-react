# Booki

Ce projet est une application web de recherche d'hébergements et d'activités en fonction des lieux, offrant une vue détaillée des logements et des options populaires dans chaque destination.

## Fonctionnalités principales

- **Page d'accueil** : Présente une vue d'ensemble avec une barre de recherche pour les lieux (pré-remplie pour démonstration).
- **Hébergement par lieu** : Affiche une liste d'hébergements disponibles dans une destination spécifique.
- **Les plus populaires** : Une section mettant en avant les hébergements les plus populaires dans divers lieux.
- **Activités en fonction du lieu** : Propose des activités recommandées basées sur le lieu sélectionné.
- **Responsive** : Le site est entièrement responsive et adapté aux écrans de toutes tailles.

## Technologies utilisées

- **React** - Framework JavaScript pour construire l'interface utilisateur.
- **CSS** - Stylisation de l'interface avec des styles personnalisés.
- **GitHub Pages** - Pour héberger l'application en ligne.

## Installation et lancement local

1. Cloner le dépôt :
   Clone ce dépôt sur ta machine locale :
   ```bash
    git clone https://github.com/votre-utilisateur/nom-du-repository.git
   ```
2. Installez les dépendances :

   ```bash
    npm install
   ```

3. Lancez l'application en mode développement :
   ```bash
    npm start
   ```
